HTML 的基本結構

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
<h1>This is a Heading</h1>
<hr>
<p>This is a<br> paragraph.</p>
</body>
</html>
要注意的是
<html>為網頁的根源
<head><title></title><head>可以呈現頁面名稱
<body></body>可以呈現在網頁的內容
<h1></h1>可以表示標題，最多到<h6></h6>
<p></p>可以表示內文
大多數的tag為成雙成對，<tag>內容</tag>，中間所包含的內容是根據tag的屬性來呈現
<hr>為一條分隔線
<br>為文字於此要跳下一行
程式圖例
以瀏覽器讀取程式後結果
各種其餘的HTML ELEMENT
<a></a> 引用
<img> 插入圖片
<pre></pre> 維持內文格式輸出
<meta> 定義head中metadata的相關屬性使用
以ATTRIBUTES來增加對ELEMENT的描述
對<html<的屬性描述
lang 範例：
<html lang="en-US">
設定此網頁的語言
對<meta>的屬性描述
<meta charset = "UTF-8">
設定網頁編碼方式為UTF-8
對<a>的屬性描述
herf 範例：
<a href="https://www.w3schools.com">This is a link</a>
縮寫為hypertext reference，也就是超連結
對<img>的屬性描述
src、width、height、alt 共同範例：
<img src="img_girl.jpg" width="500" height="600" alt="圖說">
src也就是source會有絕對路徑及相對路徑之分，如果圖片來源在相同網址之下的話可以省略重複的部分，此為相對路徑，如果沒有擇要將完整的網址輸入在="中"，
width為圖片寬度
height為圖片長度
alt為圖片無法顯示時之說明
對<h1><p>的屬性描述
title 範例：
<p title="I'm a tooltip">This is a paragraph.</p>
