HTML之menu製作關鍵想法紀錄:

1.以一個 <div> 包覆 <a></a> 或 <li><a></a></li>，由於會使用float: left; 的屬性在 <a> 或 <li> 上，會使 <div> 喪失高度，令背景顏色消失，這時可以另外於 <div> 上設定 height 使顏色顯現。
2.在適用於手機螢幕的模式下，為了使所有連結能順利出現，要於之前設定 <a> 是display: block; 絕對要附加有高度的參數，例如: height、line-height、padding …
3.要使float: right; 的link能順利於手機螢幕的模式下與其他link 對齊，可以使用 text-align: center，並在 @media 中加上 text-align: left、right、center，即可對齊。
4.float: left，float: right，不會超過前面 block-level element，只會貼齊，但是下一個未float的物件不論是 inline-block 或是 block 在寬度符合的情況下都會越過他向上貼齊，此時若不想這樣排列可以讓它一起 float 或是再在已經 float 的物件外加一層有設定高度的 block 來撐出距離。
5.要小心就算在float的物件加上 width: 100%; 若上方被撐出足夠的空白處(若用已float的物件中的物件撐出的)，inline-block 可以越過它(100%的物件)塞上去。
6.對 float 的物件進行 position: relative 宣告是有作用的，可以設定它漂浮的參照點。
